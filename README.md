# Linear Regression Project

## Description

Ce projet applique la régression linéaire en Python pour analyser et prédire le temps d'écran quotidien (screen time) basé sur différentes habitudes d'utilisation du smartphone.

## Objective

L'objectif principal est d'étudier la relation entre l'utilisation du téléphone (réseaux sociaux, jeux, sommeil, etc.) et le temps total passé sur écran par jour.

## Dataset

Le dataset contient des informations sur l'utilisation du smartphone, comme :

* age
* social_media_hours
* gaming_hours
* sleep_hours
* notifications_per_day
* app_opens_per_day
* weekend_screen_time

## Tools & Libraries

Ce projet a été réalisé avec :

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Steps

1. Data loading
2. Data cleaning
3. Exploratory Data Analysis (EDA)
4. Data visualization
5. Model training (Linear Regression)
6. Model evaluation

## Model

Le modèle utilisé est une régression linéaire simple implémentée avec Scikit-learn pour prédire le temps d'écran à partir des variables explicatives.

## Model Performance

* R² Score : à compléter
* Mean Absolute Error : à compléter

## Results

Le modèle montre qu'il existe une relation entre certaines variables (comme l'utilisation le week-end et l'utilisation des applications) et le temps d'écran quotidien.
Le modèle de régression permet de faire des prédictions raisonnables sur le screen time.
